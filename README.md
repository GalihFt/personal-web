## Projects

### Purpose

Halaman untuk menunjukkan hasil kerja nyata dan cara berpikir analitis.

### Suggested Project Card Format

Setiap proyek bisa berisi:

- Project title
- Background / problem
- Goal
- Your role
- Dataset / source
- Tools used
- Key insights
- Impact or outcome

### Example Projects

#### 1. Sales Performance Dashboard

Membangun dashboard interaktif untuk memantau revenue, top products, regional performance, dan monthly trends.

**Role:** Data Analyst  
**Tools:** SQL, Excel, Power BI  
**Output:** Dashboard untuk mempermudah monitoring performa penjualan bulanan  
**Key Insight:** Penjualan tertinggi datang dari segmen produk tertentu, tetapi growth paling cepat justru muncul di kategori yang sebelumnya kurang diperhatikan.

#### 2. Customer Segmentation Analysis

Melakukan segmentasi pelanggan berdasarkan frekuensi transaksi, total belanja, dan kategori produk favorit.

**Role:** Data Analyst  
**Tools:** Python, Pandas, Scikit-learn, Tableau  
**Output:** Segmentasi pelanggan untuk membantu strategi promosi  
**Key Insight:** Terdapat kelompok pelanggan loyal bernilai tinggi yang responsif terhadap penawaran bundling, sementara pelanggan baru lebih responsif terhadap diskon awal.

#### 3. Marketing Campaign Effectiveness Report

Mengevaluasi performa kampanye pemasaran digital berdasarkan conversion rate, cost per acquisition, dan channel performance.

**Role:** Reporting and Analysis  
**Tools:** Excel, Google Sheets, Looker Studio  
**Output:** Laporan evaluasi channel dan rekomendasi optimasi budget  
**Key Insight:** Channel tertentu memiliki reach tinggi tetapi efisiensi rendah, sehingga alokasi budget perlu dialihkan ke channel dengan conversion lebih stabil.

## Notes for Adapting This Mock

Saat kamu mengubah mock ini jadi portofolio pribadi, kamu bisa:

- Mengganti nama, role, dan deskripsi dengan profilmu sendiri
- Memindahkan `Education` ke `About` atau `Experience` sesuai kebutuhan
- Menyesuaikan jumlah proyek unggulan agar tidak terlalu banyak
- Menggabungkan `Achievements` dan `Certificates` jika ingin lebih ringkas
- Mengubah tone tulisan menjadi lebih formal atau lebih personal

## Recommended Principle

Untuk portofolio yang kuat, usahakan setiap halaman punya fungsi yang jelas:

- `Home` untuk menarik perhatian
- `About` untuk membangun koneksi dan konteks
- `Experience` untuk menunjukkan perjalanan
- `Projects` untuk membuktikan kemampuan
- `Achievements` untuk memperkuat kredibilitas
- `Contact` untuk mendorong aksi
